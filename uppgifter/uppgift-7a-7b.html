Det finns olika element/taggar att använda i text i HTML för lägga fokus på specifika saker.

Man kan betona ett ord med <em>, som vanligtvis gör texten kursiv.

Om man vill ange att något är viktigt kan man använda sig av <strong>. Vanligtvis får den texten fetstil.

Ibland kan man få för sig att använda kursiv text med hjälp av <i>. Detta skiljer sig från <em> då det används för att få en "alternativ röst eller känsla". Vill du betona? Använd em. Vill du bara ha utseendet? Använd CSS. Är det ett speciellt ord/fackterm som ska förklaras? Använd dfn (som står för definition). Har du en annan orsak så är <i> ett bra val.

Det finns ytterligare ett element: <b>. Detta används när man vill att texten ska få uppmärksamhet, men är inte betonad eller viktig. Även om det tidigare har haft betydelsen fetstil (bold) så är det numera en fråga om varför man vill att texten ska ha fetstil.

Vid förkortningar kan man använda <abbr> pga att det går snabbare att skriva saker. Använd title-attributen för att skriva en förklaring till förkortning första gången du använder det så folk vet vad du menar.

Det finns även ett element för att skriva tid och datum. Då används <time>. På nyårsafton smäller raketerna. Attributet som användes här är datetime.

I detta dokument har <em> gjorts om så att det visas med icke-kursiv fetstil, samt att <strong> visas med icke-fet kursiv stil.

Själv tycker jag att semantiken inte riktigt ändras när man gör om <em> till icke-kursiv fetstil och <strong> till icke-fet kursiv stil. Anledningen till detta är för att texten fortfarande sticker ut från mängden och man förstår att det är något viktigt som nämns.
